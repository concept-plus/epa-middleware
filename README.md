# Overview

This is the back-end app for the EPA effort. It's a node.js app using express.js.

# Installation

**Docker**  
  
_See the [Docker hub registry](https://registry.hub.docker.com/u/conceptplus/epa-rfi/)_

# Running

From the project root directory run:

    > npm install
    > node app.js
